Responsive Typography

This text uses clamp() function to scale smoothly between minimum and maximum sizes based on viewport width.

The clamp() CSS function takes three parameters: minimum value, preferred value (often using viewport units), and maximum value.

Example: font-size: clamp(1rem, 4vw, 2.5rem);
